# Relationale Felddynamik (Relational Field Dynamics)

Autor: Enno Matthiesen

Ein theoretischer Rahmen zur Erforschung relationaler Raumzeit, Gravitation, Kosmologie und Quantenstruktur.

Die Arbeiten entwickeln einen relationalen Ansatz physikalischer Realität auf Basis von acht Axiomen.  
Innerhalb dieses Ansatzes werden **Sinn, Energie und Kohärenz** als grundlegende organisierende Prinzipien des Universums betrachtet.
